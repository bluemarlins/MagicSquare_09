**테스트 목적**

**테스트 기능**

**입력값**
```python

    [16,  3,  2, 13],
    [ 5, 10, 11,  0],   # 빈칸 (1, 3)
    [ 9,  6,  7, 12],
    [ 4, 15, 14,  0],   # 빈칸 (3, 3)
]
```

| 순번 | 단계 | 절차 | 예상 결과 | 실제 결과 | 합격여부 |

| 2 | When | `find_blank_coords(matrix)` 호출 | 함수 정상 실행 | | |
| 3 | Then | 반환값이 `[(1, 3), (3, 3)]`임을 단언 | `[(1, 3), (3, 3)]` | | |

**테스트 환경**
Python 3.10+, pytest 7.x

**전제 조건**
- `matrix`가 `list[list[int]]` 형식
- 크기 4×4, 셀 값 ∈ {0, 1, …, 16}

**성공 / 실패 기준**
- 성공: `[(1, 3), (3, 3)]` 정확히 반환
- 실패: 다른 좌표 반환 또는 예외 발생
